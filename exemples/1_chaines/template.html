Exemples de remplacements de chaînes

Remplacement de contenus entre commentaires

Le contenu que vous pouvez lire dans ce template sera remplacé par celui défini dans le fichier PHP.

Remplacement de contenu entre hashes

Dans l'image suivante, l'attribut src est entre hashes. Dans le fichier rendu, on affichera l'image désirée.

Il est aussi possible de remplacer n'importe quel contenu entre hashes : la balise <title> de la page est elle aussi remplie avec des hashes.

L'attribut complate

L'exemple précédent avait l'inconvénient de produire une image cassée dans le fichier de template. Pour y remédier, il suffit de laisser une image par défaut dans l'attribut src, et d'ajouter un attribut complate écrit de la manière suivante : <img src="defaut.png" complate="src=src2">

Il est possible de créer plusieurs attributs à la fois en les mettant à la suite dans l'attribut complate, séparés par des virgules. Les attributs inexistants seront créés. L'imagine suivante aura une source et un titre attribués par Complate.
