Affichage conditionnel

La fonction setData() de Complate ajoute deux clés booléennes par élément XXX : IS_XXX et IS_NOT_XXX.

Dans Complate, un booléen se comporte de la manière suivante :

  • true : ne modifie pas le template, en laissant les commentaires HTML.
  • false : supprime le contenu ainsi que les commentaires HTML du template

Exemple avec un fil de commentaires

Dans l'exemple suivant, les boutons et le message d'erreur sont entourés de balises <!-- IS_COMMENTAIRES --> et <!-- IS_NOT_COMMENTAIRES -->. Par défaut, les commentaires étant présents, leurs valeurs respectives sont true et false. Le script détecte la présence du paramètre _GET nocomment, désactive les commentaires (comments = false) et par conséquent inverse les valeurs de is_commentaires et is_not_commentaires.

Comme vous pouvez le constater dans le code source du template, il n'est pas nécessaire d'entourer la paire de balises <!-- COMMENTAIRES --> de balises <!-- IS_COMMENTAIRES -->, l'absence de contenu dans la première paire de balises entraîant automatiquement leur disparition.
